Keyword reference
msg
Human-readable alert description. Shown in logs and SIEM.
content
Match literal bytes in payload. Use |XX XX| for hex. Multiple content keywords are ANDed.
pcre
Perl-compatible regex. Modifiers: i=nocase, s=dotall, H=http_header, U=http_uri.
nocase
Make preceding content match case-insensitive.
fast_pattern
Tell the engine to use this content for the fast-pattern matcher (performance hint).
classtype
Categorizes the rule. Affects default priority if priority is not set.
sid / rev
Unique rule identifier + revision. SIDs 1–999999 reserved for official rules. Use 9000000+ for custom.
flow
Direction context: to_server,established for client→server on existing TCP sessions.
